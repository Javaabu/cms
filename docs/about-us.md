## About Javaabu

[Javaabu](https://javaabu.com) is a web design agency based in the Maldives.

As we dive into creating top-notch projects for our clients, we stumble upon nifty solutions that make life easier for developers like us. When we realize we've cracked the code to common problems, we tidy it up and share the love.

## About Javaabu CMS

The Javaabu CMS package was born from our experience building multiple content-driven websites and applications for clients in the Maldives. We needed a flexible, developer-friendly CMS that could handle:

- Multiple languages (especially Dhivehi/Thaana script)
- Complex content types with varying features
- Hierarchical category structures
- Role-based permissions
- Modern content editing experiences

Rather than reinventing the wheel for each project, we extracted our battle-tested CMS components into this reusable package. It's now used in production by several government and private sector websites in the Maldives.

## Our Philosophy
